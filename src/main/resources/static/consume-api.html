Consume this Flight-log API

Its very easy! It has two rest end points.
Acces them as shown below.
Just replace the <ipaddress-in-address-bar> by ipaddress of this page

POST
$ curl -H "Content-Type: application/json" -X POST -d '{"sNo":"0.03", "pm_iAddrModeS":"2", "pm_dLatitude":"52.308928", "pm_dLongitude":"10.373013", "pm_dAltitudeMSL":"500.000000","pm_dRoll": "0.001500", "pm_dPitch":"0.180390", "pm_dHeading":"1.396260", "pm_dGroundTrack":"1.444650", "pm_dAirSpeed":"66.902228", "pm_dGroundSpeed":"92.341750", "pm_dVerticalSpeed":"0.005334"}' http://<ipaddress-in-address-bar>:8080/post

GET
$ curl -X GET http://<ipaddress-in-address-bar>:8080/get/0.03/


Consume from a proper java client
We had build two java client for consume this API. One is producer. Its use to send data to API (POST). Another one is consumer. Its use to consume the data from API(GET). Click here to download those jar files.

Producer
$ java -jar producer.jar TraficServerLog.txt

Consumer
$ java -jar consumer.jar
